# hangul

Hangul worksheet generator.

The goal is to learn hangul in context.

If copying the entire grid of Hangul works for you, go for it, lots of pre-existing free resources.

## Installation

``
uv sync
``

Download this font https://fonts.google.com/noto/specimen/Noto+Sans+KR

and unzip it to `google_font`

## Usage

Ask an LLM to generate Sentence objects using the pattern in `hangul\__main__.py`

```bash
make run
```

Use an even number of sentences so you can print on both sides without wasting paper.
